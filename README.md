### Flutter + NestJs 스터디용
## Clone -> 폴더 이동
* `git clone (저장소 주소)`
* `cd ./fullstack-api-project`

## Bash 터미널 1 : Back
* `cd ./nestjs_backend_api_ys`
* `npm install`
* `npm run build`
* `npm run start:dev`

## Bash 터미널 2 : Front
* `cd ./flutter_frontend_ys`
* `flutter pub get`
* `flutter create .`
* `flutter run`
